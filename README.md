# AI Contractor Mastermind Phase Assessment

Static Vercel-ready questionnaire site for routing students into the highest-value phases and modules from the AI Contractor Mastermind.

## Deploy

Run this from the `contractor-assessment` folder:

```bash
vercel
```

The site has no build step and can also be served locally with any static file server.

## Local Preview

```bash
python3 -m http.server 4173
```

Open `http://localhost:4173`.
